# Library_Management_System

📚 Library Management System (Python + Streamlit)

A simple and interactive Library Management System (LMS) built using Python, Streamlit, and JSON.
This project allows students to create accounts, issue books, return books, update details, and delete accounts—all through a clean web interface.

🚀 Features
✅ Account Management

Create new student accounts

Auto-generate unique Student ID

Update existing details

Delete accounts

📖 Book Management

Issue books (Max limit: 5)

Submit/return books

Prevent issuing more books than allowed

Prevent submitting more books than issued

⏱ Time Tracking

Store last book issue time

Store last book submission time

🗂 Database

Uses data.json as a lightweight local database

Auto-saves after each operation

🧩 Tech Stack
Component	Technology
Frontend	Streamlit
Backend	Python (OOP)
Database	JSON
UI Framework	Streamlit Widgets
